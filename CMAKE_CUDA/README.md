# CMAKE_CUDA

## 使い方

```
$ pwd
/home/user/CMAKE_CUDA
$ cd cmake-build/
$ pwd
/home/user/CMAKE_CUDA/cmake-build
$ cmake ..
$ make
$ ./Main 
Hello
OK
```

## ゴミ処理
```
$ ./clean.sh
```
